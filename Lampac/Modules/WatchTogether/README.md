# WatchTogether

Модуль синхронизированного просмотра видео для Lampac. Обеспечивает WebSocket-синхронизацию состояния плеера (play, pause, seek, смена серий) между клиентами Lampa и анонимными зрителями через веб-браузер (Vidstack Player).

## Назначение

- Обработка WebSocket-соединений происходит в пространстве имён `/nws`.
- Реализует расчёт RTT-пинга и корректировку `playbackRate` для сглаживания рассинхронизации.
- Позволяет подключаться анонимным пользователям через веб-интерфейс (`/watch_together/{id}`).
- Поддерживает установку пароля на комнату, передачу прав хоста и фоновую очистку (GC) пустых комнат.
- Встроенный веб-плеер использует HLS (Vidstack) и уникальную генерацию имён гостей.

## HTTP (обзор)

| Маршрут                      | Описание                                                                               |
| ---------------------------- | -------------------------------------------------------------------------------------- |
| `GET /watchtogether.js`      | Плагин для клиента Lampa. Инициализирует UI, перехватывает события встроенного плеера. |
| `POST /watchtogether/create` | Создание комнаты (с поддержкой пароля). Возвращает `id` комнаты.                       |
| `GET /watchtogether/list`    | JSON-список активных комнат (с проверкой флага `isProtected`).                         |
| `GET /watch_together/{id}`   | Анонимный веб-плеер. Запрашивает пароль через HTML-форму при наличии защиты.           |

## Конфигурация

Секция в `init.conf`: **`WatchTogether`**.

| Поле                       | Смысл                                                                                                                          |
| -------------------------- | ------------------------------------------------------------------------------------------------------------------------------ |
| `max_rooms_total`          | Глобальный лимит комнат на сервере (по умолчанию **500**).                                                                     |
| `max_rooms_per_user`       | Лимит комнат для одного пользователя (по умолчанию **5**).                                                                     |
| `gc_empty_timeout_minutes` | Удаление пустой комнаты через N минут (по умолчанию **30**).                                                                   |
| `gc_max_lifetime_hours`    | Удаление любой комнаты через N часов (по умолчанию **12**).                                                                    |
| `ws_ping_interval`         | Интервал пинга WebSocket в секундах (по умолчанию **20**).                                                                     |
| `allow_anonymous`          | Разрешает доступ к веб-плееру. Если `false`, WebSocket-соединения с `uid` формата `web_*` блокируются (по умолчанию **true**). |
| `web_guests_interactive`   | Разрешает веб-гостям управлять воспроизведением (по умолчанию **true**).                                                       |
