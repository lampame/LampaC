# AiLiberty

Онлайн-источник **AiLiberty** (парсинг сайта **`https://ailiberty.top`**).

## Интерфейс

**`IModuleLoaded`**, **`IModuleOnline`**.

## Условие (`Invoke`)

Только **`args.isanime`**. Пункт: плагин **`ailiberty`**, имя **`AiLiberty`**.

## Глобальный поиск

**`with_search.Add("ailiberty")`**.

## Конфигурация

Секция в `init.conf`: **`AiLiberty`** (`OnlinesSettings`).

По умолчанию: **`displayindex = 110`**, **`stream_access = apk,cors,web`**, **`httpversion = 2`**.

## Подпись качества

**`OnlineApiQuality`**: при **`e.balanser == "ailiberty"`** → **` ~ 1080p`**.

## HTTP

| Маршрут                         | Назначение                                                               |
| ------------------------------- | ------------------------------------------------------------------------ |
| **`lite/ailiberty`**            | Основная выдача (поиск и получение списка серий).                        |
| **`lite/ailiberty/video.m3u8`** | Ленивая выдача ссылок (Lazy Loading) на потоки в момент воспроизведения. |

## Особенности (Архитектура)

- **Протухающие ссылки**: Балансер `ailiberty.top` использует одноразовые токены, срок жизни которых крайне мал.
- **Ленивое проксирование потоков (Lazy Loading)**: В связи с тем, что ссылки быстро протухают, этот модуль не кеширует плейлисты и не отдаёт прямые HLS-ссылки клиенту в момент загрузки списка серий. Вместо этого в Lampa передаётся локальная ссылка на внутренний роут `/lite/ailiberty/video.m3u8`. В момент нажатия "Play", Lampac перехватывает запрос, скачивает актуальную страницу сериала, извлекает свежую ссылку на нужную серию в выбранном качестве и отдаёт клиенту HTTP-редирект (302). Это позволяет смотреть сериалы "запоем" без получения ошибки 403 при автоматическом переходе на следующую серию.
